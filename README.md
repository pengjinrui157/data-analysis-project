# 销售数据分析项目（2019年度1-8月复盘）

## 项目背景
基于2019年1-8月某公司的销售数据（共40,514条原始记录），使用 Python 完成数据清洗，MySQL进行查询，Power BI 制作可视化看板，最终输出《销售数据分析与业务洞察》报告。项目旨在通过数据分析发现业务问题，提出可落地的经营策略。

## 技术栈
- **Python**：pandas（数据清洗、聚合）、matplotlib（帕累托图、趋势图）
- **SQL**：MySQL（基本查询）
- **Power BI**：销售看板（省份毛利、商品贡献、月度趋势等）
- **数据格式**：CSV, Excel

## 文件说明

| 文件 | 说明 |

| `*.png` | 分析结果图片（月度销售、帕累托、Top10省份等） |

## 分析结论摘要

### 整体KPI（2019年1-8月）
- **总销售额**：72,630,000 元
- **总毛利**：43,700,000 元
- **整体毛利率**：60.17%

### 销售趋势
- 1-8月销售额稳步上升，**6月达到峰值**（年中大促+夏季消费推动）
- 7-8月正常回落，但仍保持高位

### 品类贡献
- **零食类**：毛利贡献率最高，是核心利润引擎
- **生鲜类**：单品毛利低，但销量大、频次高，是引流基石

### 区域表现
- **毛利Top3省份**：广东、浙江、福建（沿海地区市场强势）
- **低毛利风险省份**：新疆、甘肃、青海（需重点关注或收缩）

### 商品结构（帕累托分析）
- **约20%的核心商品贡献了80%的利润**（二八定律显著）
- Top20商品是毛利的主要来源，建议资源向头部集中，精简长尾SKU

## 业务建议（基于数据洞察）
1. **产品策略**：将高利润商品（格力高百奇、三全馒头等）放置在主通道端架，组合促销
2. **门店优化**：评估新疆、甘肃、青海等低毛利省份门店，考虑关停或缩小规模
3. **定价调整**：针对高销量低利润商品（如微爽日用245mm）提价或更换供应商
4. **季节备货**：3-4月提前加大零食、生鲜备货，迎接5-6月高峰；7-8月适度控量
